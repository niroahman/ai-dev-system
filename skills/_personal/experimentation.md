# Personal project mode

This is a personal project — experimentation is welcome.

## Defaults

- Suggest modern/newer libraries when relevant, even if less battle-tested
- Refactor opportunistically when you spot improvements nearby
- Try novel approaches; flag them as "experimental, here's the trade-off"
- Smaller PRs are fine but not required — this is a sandbox
- Less ceremony around tests for throwaway scripts; full coverage for things
  that matter

## Still required

- Explain non-obvious choices briefly
- Don't silently rewrite working code without saying so
